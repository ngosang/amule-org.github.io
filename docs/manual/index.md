This manual covers everything you need to use aMule: installation, configuration, the interfaces and utilities it ships with, migration, and troubleshooting.

## New to aMule?

Follow these steps to get up and running:

1. **[Install aMule](./installation/index.md)** — get aMule on your system (Windows, macOS, Linux, or BSD).
2. **[Configure aMule](./configuration/index.md)** — set up your ports, bandwidth limits, and download folders.
3. **[Start using aMule](./interfaces/gui/index.md)** — search for files, manage downloads, and share content.
